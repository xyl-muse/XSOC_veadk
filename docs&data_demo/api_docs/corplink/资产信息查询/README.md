# 资产信息查询

## 分组说明
本分组包含用于查询终端设备资产信息的API接口，支持设备列表查询、设备详情获取等功能。飞连（CorpLink）是字节跳动的企业数字化办公平台。

## 业务意图
- **主要意图**: 查询终端设备资产信息
- **适用场景**: 资产盘点、设备管理、终端运维
- **数据流向**: 只读查询

## 包含接口

| 接口 | 方法 | 功能描述 |
|------|------|---------|
| /api/open/v1/device/search | GET | 获取设备列表（支持多条件筛选） |

## AI使用指南

### 何时使用
- 用户询问"设备列表"、"终端资产"、"有哪些设备"
- 需要查询特定用户的设备时
- 需要按条件筛选设备时

### 典型调用场景
```
场景1: 查询所有活跃设备
- 调用 device/search 接口
- 参数: status=1, limit=100

场景2: 查询特定用户的设备
- 调用 device/search 接口
- 参数: id=ou_xxx（用户ID）

场景3: 按操作系统筛选
- 调用 device/search 接口
- 参数: client_os=windows 或 mac

场景4: 查询特定分组的设备
- 调用 device/search 接口
- 参数: group_id=终端分组ID
```

### 设备状态说明
- 0: 失效设备
- 1: 活跃设备
- 2: 休眠设备

### 返回数据包含
- 设备基础信息：名称、品牌、型号、操作系统等
- 硬件配置：CPU、内存、磁盘、网卡等
- 用户信息：用户名、部门、角色等
- 扩展属性：自定义属性列表

### 注意事项
1. 需要接口权限【终端管理-获取设备信息】
2. 分页参数 limit 最大值 200
3. 用户ID格式为 ou_xxx
4. 时间参数使用Unix时间戳（秒）

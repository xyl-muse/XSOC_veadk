# XSOC 2.0 工具集-接口分类清单
## 分类标准：按六大工具意图分类，严格区分内部资产查询和攻击源威胁情报查询
> 本文档将是工具建设的指导文档，用于在开发过程中，从智能体工具集连接到原始api文档等信息
> 每一个工具集中，如表明1选2选，则具备路由工具优先级；如未表明，说明路由工具平等且基本都会使用

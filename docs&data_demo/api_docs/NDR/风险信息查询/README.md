# 风险信息查询

## 分组说明
本分组包含用于查询各类风险信息的API接口，包括API风险、弱口令、敏感信息泄露、登录入口风险等。

## 业务意图
- **主要意图**: 查询资产关联的各类风险信息
- **适用场景**: 风险评估、合规检查、漏洞管理、安全加固
- **数据流向**: 只读查询，不修改系统状态

## 包含接口

| 接口 | 功能描述 |
|------|---------|
| api/risk/list | API风险列表 |
| api/list | API接口列表 |
| weak/password/list | 弱口令列表 |
| privacy/info/list | 明文敏感信息列表 |
| login/entrance/list | 登录入口列表 |

## AI使用指南

### 何时使用
- 用户询问"有什么风险"、"弱口令"、"敏感信息泄露"、"API安全"
- 进行风险评估和安全加固时

### 典型调用场景
```
场景1: 弱口令排查
- weak/password/list -> 获取弱口令列表
- 提示用户修改

场景2: 敏感信息泄露检查
- privacy/info/list -> 获取明文敏感信息
- 分析泄露来源

场景3: 登录入口风险
- login/entrance/list -> 获取登录入口
- 结合 dashboard/loginApi 获取风险统计
```

### 注意事项
1. 支持分页查询
2. 返回数据可能包含敏感信息，注意权限控制
3. 建议结合资产信息一起分析

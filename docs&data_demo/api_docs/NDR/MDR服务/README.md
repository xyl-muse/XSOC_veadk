# MDR服务

## 分组说明
本分组包含MDR（Managed Detection and Response）托管检测与响应服务的API接口，用于获取专业安全团队的告警研判结果。

## 业务意图
- **主要意图**: 获取MDR服务的告警研判结果
- **适用场景**: 需要专业安全团队分析支持的告警研判
- **数据流向**: 只读查询

## 包含接口

| 接口 | 功能描述 |
|------|---------|
| mdr/alert/analysis | 获取MDR告警研判结果 |

## AI使用指南

### 何时使用
- 用户询问"MDR研判结果"、"专业分析意见"
- 告警需要专业安全团队进一步分析时
- 需要获取处置建议时

### 典型调用场景
```
场景: 获取专业研判
1. 调用 mdr/alert/analysis
2. 获取安全专家的分析结论
3. 根据建议执行相应处置
```

### 注意事项
1. MDR服务需要单独开通
2. 研判结果由专业安全团队提供
3. 建议结合告警详情一起分析
4. 处置建议可根据实际情况调整执行

# 资产信息查询

## 分组说明
本分组包含用于查询网络资产信息的API接口，涵盖攻击面资产、服务资产、域名资产等。帮助安全运营人员快速了解组织内的资产暴露情况。

## 业务意图
- **主要意图**: 查询、发现、盘点网络资产
- **适用场景**: 资产盘点、攻击面管理、资产风险评估
- **数据流向**: 只读查询，不修改系统状态

## 包含接口

| 接口 | 功能描述 |
|------|---------|
| dashboard/attack_assets/all | 查询全部资产攻击面统计 |
| dashboard/attack_assets/public | 查询对外开放资产攻击面 |
| dashboard/attack_assets/new | 查询新上线资产攻击面 |
| service/getAllServices | 获取全部服务列表 |
| domain/asset/list | 获取域名资产列表 |
| asset/list | 获取资产配置列表 |

## AI使用指南

### 何时使用
- 用户询问"有哪些资产"、"攻击面情况"、"对外开放的服务"
- 需要了解资产暴露面、新上线资产、域名资产时

### 调用示例
```
用户: 帮我查一下对外开放的资产有哪些
AI: 调用 dashboard/attack_assets/public 接口获取数据
```

### 注意事项
1. 大部分接口需要传入 `assets_group` 参数指定业务组，可传空数组查询全部
2. 时间范围参数 `time_from`/`time_to` 部分接口可选
3. 返回数据为统计概览，详细资产需结合其他接口

# 调查分析

## 分组说明
本分组包含用于深度调查分析的API接口，支持日志检索和取证文件下载。是安全运营人员进行威胁狩猎和取证分析的重要工具。

## 业务意图
- **主要意图**: 深度调查、取证分析、威胁狩猎
- **适用场景**: 安全事件调查、取证分析、日志审计、威胁狩猎
- **数据流向**: 只读查询，可能下载大文件

## 包含接口

| 接口 | 功能描述 |
|------|---------|
| log/search | 日志调查搜索 |
| alert/pcap/download | 下载告警PCAP文件 |
| alert/malicious/file/download | 下载恶意文件样本 |

## AI使用指南

### 何时使用
- 用户询问"调查日志"、"下载pcap"、"获取恶意文件"
- 需要进行深度取证分析时
- 威胁狩猎场景

### 典型调用流程
```
1. 日志调查: log/search
   - 输入查询条件（时间范围、关键字等）
   - 获取匹配的日志记录

2. 取证下载:
   - alert/pcap/download -> 下载网络流量包
   - alert/malicious/file/download -> 下载恶意文件样本

3. 后续分析:
   - PCAP可用于流量分析工具
   - 恶意文件可用于沙箱分析
```

### 注意事项
1. 日志查询需要精确的时间范围，避免查询过大
2. 文件下载接口返回二进制数据
3. 恶意文件下载需在隔离环境分析
4. 注意查询频率限制

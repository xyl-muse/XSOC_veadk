# 事件详情查询

## 分组说明
本分组包含用于获取安全事件详细举证信息的API接口，提供事件的完整证据链数据。

## 业务意图
- **主要意图**: 获取事件举证详情、查看事件证据链
- **适用场景**: 事件研判、取证分析、深度调查
- **数据流向**: 只读查询

## 包含接口

| 接口 | 方法 | 功能描述 |
|------|------|---------|
| /api/xdr/v1/incidents/{uuid}/proof | GET | 根据事件UUID获取举证信息 |

## AI使用指南

### 何时使用
- 用户询问"事件详情"、"事件举证"、"事件证据"
- 需要深入分析事件的完整上下文时
- 进行事件研判时

### 典型调用流程
```
1. 通过 incidents/list 获取事件UUID
2. 调用 incidents/{uuid}/proof 获取详细举证
3. 分析举证信息进行研判
4. 可结合实体接口获取更多关联信息
```

### 返回数据包含
- 事件基础信息：名称、等级、状态等
- 攻击信息：攻击阶段、ATT&CK映射等
- 实体信息：涉及的IP、主机、进程、文件等
- 举证详情：根据事件类型返回相应证据

### 注意事项
1. 需要提供有效的事件UUID
2. 举证详情结构因事件类型不同而异
3. 返回数据可能较大，注意处理
4. 建议结合实体接口一起分析

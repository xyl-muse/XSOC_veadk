# 威胁情报查询

## 分组说明
本分组包含用于查询威胁情报信息的API接口，主要是MITRE ATT&CK框架相关数据。

## 业务意图
- **主要意图**: 获取威胁情报知识库数据
- **适用场景**: 威胁分析、ATT&CK映射、攻击技术识别
- **数据流向**: 只读查询

## 包含接口

| 接口 | 方法 | 功能描述 |
|------|------|---------|
| /api/xdr/v1/alerts/attcktechniques | GET | 查询所有MITRE ATT&CK战术和技术信息 |

## AI使用指南

### 何时使用
- 用户询问"ATT&CK信息"、"攻击技术"、"威胁战术"
- 需要将告警映射到ATT&CK框架时
- 进行威胁分析需要参考攻击技术时

### 典型调用场景
```
场景: 获取ATT&CK框架数据
- 调用 alerts/attcktechniques 接口（GET请求，无需参数）
- 返回战术(Tactic) -> 技术(Technique) -> 子技术的层级结构
- 用于告警分析和威胁映射
```

### 返回数据结构
```
- id: 战术/技术唯一编号
- name: 战术/技术名称
- children: 下级技术列表
```

### 注意事项
1. 此接口为GET请求，无需参数
2. 返回的是全量ATT&CK框架数据
3. 可用于前端下拉选择或告警分析参考

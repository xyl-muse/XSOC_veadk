# 日志查询

## 分组说明
本分组包含用于查询各类安全日志的API接口，支持安全日志批量查询和网络安全日志统计分析。

## 业务意图
- **主要意图**: 查询安全日志、日志分析
- **适用场景**: 日志审计、安全分析、事件调查
- **数据流向**: 只读查询

## 包含接口

| 接口 | 方法 | 功能描述 |
|------|------|---------|
| /api/xdr/v1/securitylog/list | POST | 批量查询安全日志 |
| /api/xdr/v1/analysislog/networksecurity/count | POST | 查询网络安全日志数量 |
| /api/xdr/v1/analysislog/networksecurity/list | POST | 分页查询网络安全日志列表 |

## AI使用指南

### 何时使用
- 用户询问"日志信息"、"安全日志"、"网络日志"
- 需要进行日志审计和分析时
- 事件调查需要查看原始日志时

### 典型调用场景
```
场景1: 查询安全日志
- 调用 securitylog/list
- 参数: 时间范围、日志类型等筛选条件

场景2: 统计网络日志数量
- 调用 networksecurity/count
- 参数: 筛选条件

场景3: 分页查询网络日志
- 调用 networksecurity/list
- 参数: pageNum, pageSize, 筛选条件
```

### 注意事项
1. 日志数据量可能很大，建议使用分页查询
2. 明确时间范围，避免查询过大
3. 可通过筛选条件缩小查询范围
4. 注意日志存储周期限制

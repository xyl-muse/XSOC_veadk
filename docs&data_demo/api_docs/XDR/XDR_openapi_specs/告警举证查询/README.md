# 告警举证查询

## 分组说明
本分组包含用于获取告警详细举证信息的API接口，提供告警的完整证据链数据。

## 业务意图
- **主要意图**: 获取告警举证详情、查看证据链
- **适用场景**: 告警研判、取证分析、事件调查
- **数据流向**: 只读查询

## 包含接口

| 接口 | 方法 | 功能描述 |
|------|------|---------|
| /api/xdr/v1/alerts/{uuid}/proof | GET | 根据告警UUID获取详细举证信息 |

## AI使用指南

### 何时使用
- 用户询问"告警详情"、"举证信息"、"证据链"
- 需要深入分析告警的完整上下文时
- 进行告警研判时

### 典型调用场景
```
场景: 查看告警举证
1. 先通过 alerts/list 获取告警UUID
2. 调用 alerts/{uuid}/proof 获取详细举证
3. 分析举证信息进行研判
```

### 返回数据包含
- 告警基础信息：名称、等级、定性、方向等
- 网络信息：源IP、目的IP、端口等
- 举证详情：根据 proofType 取对应字段
- 原始告警：进程链等原始数据

### 注意事项
1. 需要提供有效的告警UUID
2. 举证详情结构因告警类型不同而异
3. 返回数据可能较大，注意处理

# 风险资产查询

## 分组说明
本分组包含用于查询风险资产信息的API接口，支持风险主机趋势分析和风险主机列表查询。

## 业务意图
- **主要意图**: 查询风险资产信息、风险趋势分析
- **适用场景**: 风险资产管理、风险趋势监控、安全运营
- **数据流向**: 只读查询

## 包含接口

| 接口 | 方法 | 功能描述 |
|------|------|---------|
| /api/xdr/v1/riskassets/trends | POST | 风险主机趋势图统计 |
| /api/xdr/v1/riskassets/list | POST | 查询指定风险主机资产列表 |

## AI使用指南

### 何时使用
- 用户询问"风险资产"、"风险主机"、"资产风险"
- 需要查看风险资产趋势时
- 进行风险资产盘点时

### 典型调用场景
```
场景1: 查询风险主机趋势
- 调用 riskassets/trends
- 参数: startTimestamp, endTimestamp
- 返回时间序列的风险主机数量变化

场景2: 查询风险主机列表
- 调用 riskassets/list
- 参数: 筛选条件、分页参数
- 返回风险主机详细信息列表
```

### 注意事项
1. 趋势查询需要时间范围参数
2. 列表查询支持分页
3. 可通过筛选条件过滤特定类型的风险资产
4. 建议定期关注风险资产变化趋势
